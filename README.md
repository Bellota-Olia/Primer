## Cómo empezar

1. Clonar el repositorio:
   git clone https://github.com/Bellota-Olia/Primer.git
2. Entrar en la carpeta:
   cd Primer

## Contribuir

Si quieres contribuir:

- Crea una rama nueva: git checkout -b mi-rama
- Haz cambios y commitea: git commit -am "Descripción de cambios"
- Sube la rama y abre un pull request en GitHub
